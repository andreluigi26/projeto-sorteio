# 🎲 Sistema de Sorteio / Rifa

Aplicação web simples para cadastro de participantes e realização de sorteios de forma prática e rápida.

## 📌 Descrição

Projeto desenvolvido com o objetivo de praticar JavaScript, focando em lógica de programação, manipulação de arrays, validação de dados e interação com o DOM.

O sistema permite cadastrar participantes, evitar nomes duplicados e realizar sorteios automaticamente.

---

## 🚀 Funcionalidades

- ✔️ Cadastro de participantes
- ✔️ Validação para evitar nomes duplicados
- ✔️ Listagem dinâmica dos participantes
- ✔️ Sorteio automático
- ✔️ Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (puro)
